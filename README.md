# spark-claude

**Agent Definition Standard** — Where practice, theory, and science converge into a unified standard for AI agent definitions.

---

## What is this?

A scientifically-grounded standard for defining AI agents, along with reference implementations for software development.

This standard emerged from the convergence of three independent sources:
- **SPARK** — Design principles derived from hands-on experience across 5 projects
- **TRAITS** — A 150-trait catalog and cognitive load theory research
- **Anthropic Research** — 6 neuroscience papers on persona mechanics in LLMs

## Quick Start

### Using Existing Agents
Copy the agent definitions from the `agents/` directory into your project.

### Creating New Agents
Follow the guide in [AGENT_DEFINITION_STANDARD.md](AGENT_DEFINITION_STANDARD.md).
