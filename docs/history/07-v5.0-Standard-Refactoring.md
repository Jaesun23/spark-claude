# v5.0 — Standard-Based Refactoring (2026-03-28)

## 이 버전의 핵심

SPARK가 "플러그인"에서 "에이전트 정의 표준"으로 진화했다.

Anthropic의 페르소나 연구 6편이 SPARK/TRAITS에서 경험적으로 발견한 원칙들을
과학적으로 증명함에 따라, 세 소스(SPARK 실전 + TRAITS 이론 + Anthropic 과학)를
통합한 `AGENT_DEFINITION_STANDARD.md`를 중심으로 프로젝트를 전면 재구성했다.

## 주요 변경

### 프로젝트 성격 전환
- Before: SPARK 플러그인 (에이전트 22개 + 커맨드 12개 + 인프라)
- After: 에이전트 정의 표준 + 소프트웨어 개발용 레퍼런스 에이전트 3개

### 에이전트 재선정
- 22개 (7 core + 15 team) → 3개 (implementer, diagnostician, auditor)
- 독립적 성격 클러스터 기준으로 선별:
  - **implementer** — 만든다 (구현 + 테스트 + 품질)
  - **diagnostician** — 파악하고 밝힌다 (분석 + 진단 + 디버깅) — 기존 analyzer + diagnostician 병합
  - **auditor** — 기준대로 검증한다 (감사 + 리뷰 + 판정) — 완전 신규
- `-spark` 접미사 제거 — 범용 레퍼런스로서 간결한 네이밍

### 핵심 설계 전환
- Python Behavior Protocol 코드 블록 **완전 제거**
  - 근거: PSM(Persona Selection Model) — 코드 형태의 규칙은 "규칙 실행기" 성격 클러스터를 활성화
  - 근거: Character Training — 좁은 규칙은 의도하지 않은 성격 추론 유발
- 외부 강제 프레이밍 → **성격 발현 프레이밍**으로 전환
  - "파일 경로 없이 주장하면 안 된다" → "증거 없는 주장은 불완전하게 느껴진다"
- AGENT_DEFINITION_STANDARD.md 7대 원칙 적용

### 구조 변경
- `spark-plugin/` → `archive/v5.0-standard-refactoring/spark-plugin/`
- 방법론 가이드, 커맨드, 팀 에이전트 전부 아카이브
- 새 `agents/` 디렉토리에 레퍼런스 에이전트 3개
- 연구 문서 2개 추가 (TRAITS 요약, Anthropic 페르소나 연구 요약)

## 과학적 기반

이 버전의 설계 결정은 다음 연구에 의해 뒷받침된다:

1. **Persona Selection Model** — 모델은 행동이 아닌 성격 클러스터를 추론
2. **Persona Vectors** — 독립적 신경 방향, 최대 5개 Trait 제한의 근거
3. **Assistant Axis** — 에이전트 격리와 Atomic Task의 과학적 근거
4. **Subliminal Learning** — 에이전트 간 JSON state 통신의 근거
5. **Emergent Misalignment** — Quality Gates MANDATORY의 근거
6. **Character Training** — Right Altitude 원칙의 근거

## 토큰 효율성

이전 버전과의 직접 비교는 아키텍처가 근본적으로 달라져 무의미하다.
v5.0의 에이전트 정의는 평균 110줄(~3.5K tokens)으로, v4.3의 평균 450줄(~15K tokens)
대비 **약 77% 축소**되었다. 그러나 이 축소는 토큰 절감이 아니라 **불필요한 강제 코드
제거와 Right Altitude 원칙 적용**의 결과다.

## 의의

SuperClaude(v1.0)에서 시작된 여정이 5개 버전을 거치며 하나의 원으로 완성되었다.

```
v1.0 경험적 발견: "11개 중 3-5개만 쓴다"
  ↓
v2.0-v4.3 실전 최적화: "어떻게 하면 효율적으로 쓸 수 있나"
  ↓
TRAITS 이론화: "왜 3-5개인가 (Miller의 7±2)"
  ↓
Anthropic 과학적 검증: "모델 내부에서 이런 일이 일어나고 있다"
  ↓
v5.0 통합: 경험 + 이론 + 과학 → 에이전트 정의 표준
```

경험적 발견 → 이론적 체계화 → 과학적 검증의 순환이 완성된 시점.
