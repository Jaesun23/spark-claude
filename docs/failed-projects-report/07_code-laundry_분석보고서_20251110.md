## 1. 프로젝트 개요

### 1.1 목적과 핵심 기능

**README.md 기반 목적 정의** (lines 1-4):
> "**2-3일 걸리던 수동 작업을 1시간 만에!**
> 더러운 코드를 깨끗하게 빨아주는 자동 오류 수정 도구"

**핵심 문제 인식**:
- **Before**: 수천 개 오류를 2-3일 동안 수동 수정
- **After**: 1시간 만에 99.98% 자동 해결
- **DNA Development Methodology v3.0의 품질 자동화 병목 현상 해결**

**핵심 기능**:
1. **7가지 품질 도구 통합**: ruff, mypy, black, isort, bandit, pydocstyle, markdownlint
2. **65개 전문 세탁기 (Washer)**: 각 오류 패턴별 최적화된 처리
3. **7단계 워크플로우**: 스캔 → 수집 → 세탁 → 네이티브 수정 → 재스캔 → 분석 → 전송
4. **프랜차이즈 모델**: 본점(HQ) - 지점(프로젝트) 구조
